# Star Wars React

Projeto para processo de seleção de vaga Front-End React.

Tecnologias necessárias para o teste do projeto:
- NodeJS

Desenvolvido utilizando as seguintes bibliotecas:
- Material UI
- React Router
- Axios

### Processo de execução 

Instalar bibliotecas node
```
npm i
 ```
 
Rodar a aplicaço
```
npm run start
 ```
